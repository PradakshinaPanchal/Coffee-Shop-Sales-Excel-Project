# Coffee Shop Sales Analysis (Excel)

## 📌 Overview
This project analyzes sales data from a coffee shop to uncover trends in revenue, customer behavior, and product performance. The goal is to turn raw transactional data into clear, actionable business insights using Microsoft Excel.

## ❓ Problem Statement
Coffee shop owners/managers often have raw sales data but lack a quick way to understand what's actually driving revenue — which products sell best, when sales peak, and how performance varies across locations or time periods. This project addresses that by building a structured analysis and dashboard from the raw sales data.

## 🎯 Objectives
- Clean and structure raw coffee shop sales data
- Identify best-selling products and categories
- Analyze sales trends by day, week, month, and time of day
- Compare performance across store locations (if applicable)
- Build an interactive dashboard for at-a-glance insights

## 🛠️ Tools & Techniques Used
- **Microsoft Excel**
- Pivot Tables & Pivot Charts
- VLOOKUP / XLOOKUP, INDEX-MATCH
- Conditional Formatting
- Data Cleaning (removing duplicates, handling blanks, text-to-columns)
- Slicers & Timelines for interactive filtering
- Dashboard design (charts, KPI cards)

## 🔍 Key Steps
1. **Data Cleaning** – Removed duplicates, fixed date/time formats, handled missing values, standardized product names.
2. **Exploratory Analysis** – Used pivot tables to summarize sales by product, category, store, and time period.
3. **Trend Analysis** – Built charts to visualize daily/monthly sales trends and identify peak hours.
4. **Dashboard Creation** – Combined KPIs (total revenue, total orders, average order value) with charts and slicers into a single interactive dashboard sheet.
